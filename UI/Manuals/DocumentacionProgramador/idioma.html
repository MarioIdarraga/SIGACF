Multilenguaje

SIGACF soporta múltiples idiomas mediante archivos externos cargados dinámicamente. Los archivos de idioma no llevan extensión (ej: Language.es-AR, Language.en-US).

Funcionamiento

  • La UI solicita el idioma actual a la SL.
  • El archivo correspondiente se carga en memoria.
  • La UI traduce textos controles, grids y mensajes de forma automática.

Ventajas

  • No requiere recompilar la aplicación para agregar un idioma.
  • El contenido es editable por administradores avanzados.
