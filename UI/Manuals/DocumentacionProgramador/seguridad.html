Seguridad

Cifrado AES

Las contraseñas se almacenan cifradas con AES utilizando una clave definida en configuración. Esto asegura que no existan contraseñas en texto plano en la base de datos o archivos.

Permisos (Composite)

El acceso a funcionalidades está controlado por Patentes (permisos unitarios) y Familias (grupos de permisos). El Service Layer evalúa permisos antes de permitir ejecutar acciones críticas.

Bloqueo de Usuario

El login incorpora un mecanismo de bloqueo tras varios intentos fallidos para prevenir ataques de fuerza bruta.
