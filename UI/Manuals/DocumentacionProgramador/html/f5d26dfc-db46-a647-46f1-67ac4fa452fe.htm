      Interfaces

      ICustomerStateRepository Define las operaciones de acceso a datos para los estados de cliente. Proporciona métodos para obtener todos los estados o uno específico por ID.
      IGenericRepositoryT Define operaciones genéricas de acceso a datos para entidades identificadas por un Guid. Permite insertar, actualizar, eliminar y obtener registros con distintos criterios de filtrado.
      IPaymentMethodRepository Define las operaciones de acceso a datos relacionadas con los métodos de pago utilizados en el sistema.
      IPayRepository Define las operaciones de acceso a datos para la entidad Pay. Permite registrar, modificar, eliminar y consultar pagos realizados en el sistema.
      IPayStateRepository Contrato para acceder a los estados de pago.
      IUserRepositoryT Define las operaciones de acceso a datos para entidades de usuario, incluyendo creación, actualización, eliminación, consultas y soporte para recuperación de contraseña.
