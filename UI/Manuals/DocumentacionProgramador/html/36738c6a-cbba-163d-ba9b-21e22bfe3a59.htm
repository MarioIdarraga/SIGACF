      BLL.Service Namespace

      [Missing <summary> documentation for "N:BLL.Service"]

      Classes

      BookingService Servicio encargado de manejar la lógica de negocio para las reservas (Bookings): - Validaciones de datos obligatorios - Validaciones de reglas de negocio (duración, duplicados, cancha existente, cliente existente) - Exposición de consultas utilizadas por la capa SL
      BookingStateService Servicio de negocio para gestionar los estados de reserva (BookingsStates). Encapsula la lógica de acceso al repositorio de BookingState.
      CustomerService Servicio que gestiona la lógica de negocio relacionada con clientes.
      CustomerStateService Servicio de negocio encargado de gestionar los estados de cliente. Actúa como capa intermedia entre la BLL y los repositorios de CustomerState.
      FieldService Servicio de negocio encargado de gestionar la lógica relacionada con las canchas (Field). Valida datos y delega operaciones al repositorio.
      LoginService Gestiona la lógica de autenticación de usuarios.
      ManagerAdminstration Servicio encargado de administrar operaciones globales de la base de datos, como la creación de backups mediante comandos SQL directos.
      PaymentMethodService Lógica de negocio para métodos de pago.
      PayService Servicio de negocio encargado de gestionar la obtención de pagos. Actúa como capa intermedia entre la BLL y el repositorio de pagos.
      PayStateService Servicio de negocio para obtener estados de pago.
      PromotionService Servicio de negocio para gestionar las promociones.
      UserService Servicio de negocio encargado de gestionar usuarios del sistema. Proporciona validación, registro, modificación y consultas de usuarios.
