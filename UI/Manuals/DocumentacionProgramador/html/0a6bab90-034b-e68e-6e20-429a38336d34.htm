      IBackupRepositoryRestoreAsync Method

      Restaura una base de datos a partir de un archivo .bak. Si el respaldo proviene de otra máquina, puede especificarse las rutas físicas para los archivos de datos (.mdf) y logs (.ldf) mediante WITH MOVE.
