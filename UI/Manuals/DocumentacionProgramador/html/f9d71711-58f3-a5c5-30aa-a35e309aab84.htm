      Classes

      BusinessException Representa una excepción controlada del negocio. Se utiliza para diferenciar errores esperados (reglas de negocio) de errores inesperados del sistema.
      CambioPasswordRequeridoException Excepción utilizada cuando un usuario debe cambiar su contraseña antes de poder continuar usando el sistema.
      CanchaInexistenteException Excepción utilizada cuando se intenta operar sobre una cancha que no existe en el sistema.
      ClienteDniDuplicadoException Se lanza cuando se intenta registrar un cliente cuyo DNI ya existe en el sistema.
      ClienteInexistenteException Excepción utilizada cuando se intenta operar sobre un cliente que no existe en el sistema.
      DatabaseAccessException Excepción que representa un error al acceder a la base de datos. Se utiliza cuando ocurre un problema de conexión, consulta o ejecución durante operaciones de persistencia.
      DuracionReservaInvalidaException Excepción que se lanza cuando la duración de una reserva no cumple con las reglas de negocio establecidas.
      PasswordIncorrectaException Excepción que se lanza cuando la contraseña ingresada no coincide con la registrada para el usuario.
      ReservaDuplicadaException Excepción que se lanza cuando se intenta registrar una reserva que ya existe para el mismo cliente, fecha u horario.
      UsuarioBloqueadoException Se lanza cuando un usuario ha superado los intentos permitidos y su cuenta se encuentra bloqueada.
      UsuarioInexistenteException Excepción que se lanza cuando el usuario solicitado no existe en el sistema o no puede ser encontrado.
