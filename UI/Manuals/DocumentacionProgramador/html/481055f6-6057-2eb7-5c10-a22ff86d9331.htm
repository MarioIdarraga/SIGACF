      IBackupRepository Interface

      Define las operaciones necesarias para realizar copias de seguridad, verificar la integridad de archivos .bak y restaurar bases de datos. Implementado por la capa DAL en repositorios específicos.

      Methods

      BackupAsync Genera un archivo de respaldo (.bak) de la base de datos y devuelve la ruta completa donde se almacenó el archivo.
      RestoreAsync Restaura una base de datos a partir de un archivo .bak. Si el respaldo proviene de otra máquina, puede especificarse las rutas físicas para los archivos de datos (.mdf) y logs (.ldf) mediante WITH MOVE.
      VerifyAsync Verifica la integridad de un archivo .bak generado. Utilizado para detectar corrupción en el respaldo antes de restaurar.
