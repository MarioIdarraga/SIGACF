3.2 Módulo de Ventas – Reservas

3.2.1 Búsqueda de Reservas

En este módulo se crean, buscan, editan y cancelan reservas; además, se registran pagos asociados.

Filtros disponibles:

  • Nro de Documento
  • Fecha de Reserva
  • Fecha de Registro

Presione Consultar Reserva para ver la grilla de resultados. El estado de la reserva se indica por colores: verde (Confirmada), rojo (Cancelada) y otros (Pendiente).

3.2.2 Registrar Reserva

  1. Desde Ventas – Clientes, modifique o confirme los datos del cliente.
  2. Vaya a Ventas – Reservas y seleccione Registrar Reserva.
  3. Complete cancha, fecha, horario y, opcionalmente, una promoción.
  4. Luego de seleccionar la fecha, presione Ver horarios para visualizar los horarios disponibles.
  5. El importe se completa automáticamente según la cancha y la promoción seleccionada.

3.2.3 Modificar Reserva

  1. Desde Ventas – Clientes, modifique o confirme los datos del cliente.
  2. Vaya a Ventas – Reservas y seleccione Modificar Reserva.
  3. Actualice los campos que desee: cancha, fecha, horario y promoción (opcional).
  4. Al seleccionar la fecha, presione Ver Horarios Disponibles para cargar los horarios libres.
