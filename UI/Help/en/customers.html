3.1 Sales Module – Customers

3.1.1 Customer Search

This module allows you to manage customers (create, edit, consult) and provides quick access to reservations.

  • Fill in one or more filters: Document Number, First Name, Last Name, Phone, E-mail, Status.
  • Click Search to list the results.
  • Use the buttons to Register Customer, Edit Customer or Register Reservation.

3.1.2 Register Customer

  1. Enter Document Number, First Name, Last Name, Phone, E-mail and Address.
  2. The Comments field is optional.
  3. Click Register. The system validates required fields and shows a confirmation message.

3.1.3 Edit Customer

  1. Search for the customer, select it and click Edit Customer.
  2. Update the desired fields: Document Number, First Name, Last Name, Phone, E-mail and Address. Comments are optional and the current status can be viewed.
  3. Click Save. The system validates the information and confirms the update.
