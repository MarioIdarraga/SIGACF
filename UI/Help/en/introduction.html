Introduction

This guide explains step by step how to use the SIGACF system to manage customers, reservations, payments, cancellations, reports and administration. It includes screenshots and standard operating procedures.

Prerequisites

  • Have an active user account in the system.
  • Access to the internet or the internal network, depending on the installation.
  • Have read and accepted the security policies of the facility.

Note: when a user account is created, a temporary password is assigned. On the first login, the user must change it.
